## Getting started

1. Gets the git project. (Auto run installation)

```
  npx react-vite-ts-boilerplate my-app
```

2. Start the dev server. And open http://localhost:3000 in your browser.

```
  yarn dev
```

3. Build your product.

```
  yarn prod
```

## Dependencies

- React
- Vite
- Typescript 
- eslint
- Prettier
